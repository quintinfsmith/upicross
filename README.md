A tiny in-console implementation of Picross

## Install
```bash
    pip install picross
```

## Run
```bash
    picross [-w <width>] [-h <height>] [-d density]
```    

Arguments:
    --help      Show this menu
    -w <N>      Set width to N spaces. Default = 20
    -h <N>      Set height to N spaces. Default = 12
    -d <N>      Set density to N  (0-1). Default = 0.6

Controls:
    h,j,k,l     Move Cursor
    x           Set Cell
    z           Block Cell
    u           Undo
    q           Quit
